# Sense Article Rich Editor QA Checklist

## 编辑与保存

- [ ] 打开一个已有 rich_html 草稿，编辑后自动保存状态能从“未保存”变成“已保存”
- [ ] 手动点击保存后，保存状态和最近保存时间更新
- [ ] 离开页面时，只有存在未保存修改时才出现提醒
- [ ] 自动保存失败时，状态区出现失败提示和重试入口
- [ ] 人为制造版本落后时，编辑页能显示冲突提示，不会静默覆盖

## 本地恢复

- [ ] 自动保存失败后刷新页面，能看到“恢复本地内容”提示
- [ ] 恢复本地内容后，正文、标题、修订说明一并恢复
- [ ] 丢弃缓存后，恢复提示消失

## 富文本能力

- [ ] 标题、段落、粗体、斜体、下划线、删除线、行内代码可正常生效
- [ ] 颜色、高亮、对齐、缩进、分割线可正常生效
- [ ] 无序/有序/任务列表切换正常
- [ ] 清除格式不会破坏段落结构

## 表格与媒体

- [ ] 插入表格时可选择行列数、表头、样式
- [ ] 表格支持增删行列、切换表头、切换样式
- [ ] 图片上传可插入，并支持 alt、caption、宽度、对齐
- [ ] 音频上传可插入，并保留标题/说明
- [ ] 视频上传可插入，并支持 poster、宽度
- [ ] 阅读态与预览态中的表格和媒体样式一致

## 粘贴与导入

- [ ] 纯文本粘贴不会带入奇怪样式
- [ ] 网页 HTML 粘贴后，段落/列表/链接等结构仍保留
- [ ] Word/WPS 粘贴不会把大量内联样式直接带进正文
- [ ] Markdown 导入对话框可以把基础 Markdown 插入到编辑器
- [ ] 粘贴本地图片文件时不会插入 base64，而是给出明确提示

## 目录与导航

- [ ] 长文目录会根据 H1-H4 自动生成
- [ ] 点击目录项能滚动定位到预览标题
- [ ] sticky toolbar 存在时，定位标题不会被遮挡

## scoped / 审阅 / compare

- [ ] scoped 修订打开编辑页后会自动高亮对应块
- [ ] compare 页能显示 rich_html 块级差异
- [ ] compare 若无差异，显示明确空状态
- [ ] 审阅页能看到发布前 blocking/warning 摘要

## legacy 兼容

- [ ] legacy 内容进入编辑页时可成功转换为 rich_html
- [ ] 转换失败时不会崩溃，而是进入只读 fallback
- [ ] legacy 内容仍可阅读、审阅、回退

## 发布前校验

- [ ] 空正文时阻止提交
- [ ] 无效内部引用时阻止提交
- [ ] 缺失媒体资源记录时阻止提交
- [ ] 图片缺 alt 时显示 warning，但不阻止提交
- [ ] 标题层级跳跃和空表格能显示 warning

## 审计与维护

- [ ] `npm run test:sense-articles` 通过
- [ ] `cd frontend && npm run build` 通过
- [ ] `npm run audit:legacy-sense-articles -- --scope=revisions --limit=50` 能输出报告
- [ ] `npm run audit:sense-article-media -- --nodeId=<nodeId> --senseId=<senseId>` 能输出报告
